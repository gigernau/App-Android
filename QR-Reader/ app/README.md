QR-Reader App
=============

Questa applicazione Android permette di leggere qualsiasi tipo di barcode o QrCode, in tempi ottimali.

Pre-requisiti
--------------

- Android SDK v23
- Android Build Tools v23.0.2
- Repository supporto Android v23.3.0
